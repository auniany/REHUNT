======================================================
  REHUNT version 1.2 (Released on October 15, 2017)
======================================================

Author: Yu-Huei Cheng <yuhuei.cheng@gmail.com>

REHUNT (Restriction Enzymes HUNTing) v1.2 is released on October 15, 2017.
This is stable version.
It is a free and open source package implemented in JAVA for
providing many useful methods for biological sequence analysis
(especially in SNP genotyping) around restriction enzymes.

======================================================
Requirements
======================================================

JAVA Runtime Environment (JRE) 8 or later.
  If you can not run REHUNT or you are getting an error message, please download and install
  the laster JRE from:
    http://www.oracle.com/technetwork/java/javase/downloads/index.html


======================================================
Database
======================================================

REBASE is download from http://rebase.neb.com/rebase/rebase.files.html
at 11. All Enzymes (parsed references) file.
The current REBASE version is 709: parsrefs.709
update: Aug 28 2017
